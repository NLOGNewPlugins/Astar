아이템 등급 정리

나무 또는 가죽 : 레어(§5)
돌 또는 사슬 : 유니크(§d)
철 : 레전더리(§6)
다이아 : 에픽(§e)


공격력 계산식 : 무기 공격력 x (1 + 공격력 % 증가)

힘 계산식 : 장비 힘의 합 x (1 + 힘 % 증가)

공뎀증 계산식 : 공격시 데미지 % 증가 + 공격시 데미지 증가량 % 추가 증가

최종 데미지 계산식 : 공격력 x (1 + 힘 ÷ 250) x (1 + 공뎀증)

크리티컬 발동시 데미지 계산식 : 최종 데미지 x 1.5 x (1 + 크리티컬 공격시 데미지 % 증가 + 크리티컬 공격시 데미지 증가량 % 추가 증가)

추가 데미지 계산식 : 최종 데미지 x (1 + 공격시 % 추가 데미지 증가량)

크리티컬 발동 확률 : 3 + 크리티컬 확률 %

% 계산법
10% 는 0.1
50% 는 0.5

합연산 사용법 예시
공격력 10 10% 20% 합연산
(1+0.1+0.2)×10=13

곱연산 사용법 예시
공격력 10 10% 20% 곱연산
(1+0.1)x(1+0.2)×10=13.2


아이템 옵션 설명

공격력 % 증가 : %를 곱연산으로 최종 데미지 계산식에 곱한다.

공격시 % 추가 데미지 : 최종 데미지를 입힌후 추가로 %를 합연산으로 최종 데미지를 한번더 입힌다.(총 2번의 데미지를 입히며 추가 데미지도 이펙트로 표시한다.)

공격시 데미지 증가량 % 추가 증가 : '공격 시 데미지 % 증가' 옵션의 수치를 합연산으로 증가시켜준다.

크리티컬 공격시 데미지 증가량 % 추가 증가 : '크리티컬 공격 시 데미지 % 증가' 옵션의 수치를 합연산으로 증가시켜준다.

크리티컬 확률 % : 크리티컬 확률을 합연산으로 % 증가시켜준다.

힘 % 증가 : 힘을 합연산으로 % 증가시켜준다.
